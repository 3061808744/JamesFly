git的使用方法 创建一个git仓库 命令：git init 把文件添加到git仓库 命令：git add 将提交的文件告诉git 命令：git commit

获取仓库的状态: git status

查看改动的地方：git diff

获取操作的历史记录 git log

回退到某个版本：git reset --hard 版本号既可以回到过去又可以穿越到之前操作过的版本

记录每一次操作的命令：git reflog

查看工作区和版本库里面最新版本的区别 ：git diff HEAD --文件名

丢弃工作区的修改 git checkout -- file

查看某个文件里面的内容:cat file

删除文件 rm file

从版本库中删除某个文件 git rm file

如果要关联一个远程库 git remote add origin 远程仓库地址

git远程仓库 git push -u origin master

推送最新修改 git远程仓库 git push origin master

从远程库克隆 git clone 远程仓库地址 git支持多种协议，包括https，但通过ssh支持原生git速度最快
